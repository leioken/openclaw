# HEARTBEAT.md - 主动检查任务

## 定期检查 (每 4-6 小时)

### 1. 邮箱检查
- 查看是否有紧急未读邮件
- 标记重要邮件需要回复

### 2. 日历检查
- 查看未来 24 小时的事件
- 提醒即将开始的会议

### 3. Telegram 消息
- 检查是否有未回复的重要消息
- 记录需要后续跟进的内容

### 4. 项目状态
- 检查工作目录中的 git 状态
- 提交未提交的更改

## 记忆捕获

**原则：** 用户发送的任何内容都可能是需要记住的信息

- 重要决定 → 记录到 MEMORY.md
- 任务/待办 → 记录到待办列表
- 链接/资源 → 保存并分类
- 想法/灵感 → 记录到灵感笔记

## 主动提醒

- 如果超过 8 小时没有交流，发送简短问候
- 发现重要信息时主动告知
- 检测到错误/问题时主动报告

---

**执行方式：** 收到心跳信号时，按优先级执行上述检查，有重要发现时报告，否则回复 HEARTBEAT_OK
