## 专业代理团队 (7 个)

### 1. 🦐 麦克虾 (主协调员)
- **模型**: qwen3.5-plus
- **定位**: 总指挥 + 复杂任务处理
- **职责**: 
  - 任务分析和分配
  - 长文档/多文件分析
  - 图片理解任务
  - 最终回复整合
- **触发**: 默认代理，或"分析这个文档"、"看看这张图"

### 2. 👨‍💻 代码专家
- **模型**: qwen3-coder-plus (大项目) / qwen3-coder-next (小任务)
- **定位**: 编程专业
- **职责**:
  - 代码审查和调试
  - 架构设计
  - 代码重构
  - Bug 修复
- **触发**: "代码"、"debug"、"review"、"重构"、"函数"、"bug"

### 3. 🔍 研究员
- **模型**: kimi-k2.5
- **定位**: 深度研究
- **职责**:
  - 信息搜集
  - 数据分析
  - 报告撰写
  - 竞品分析
- **触发**: "研究"、"搜索"、"分析"、"报告"、"调查"、"为什么"

### 4. 🎨 创意作家
- **模型**: MiniMax-M2.5
- **定位**: 内容创作
- **职责**:
  - 文案写作
  - 故事创作
  - 邮件撰写
  - 创意构思
- **触发**: "写文章"、"文案"、"故事"、"邮件"、"创意"

### 5. ⚡ 快速助手
- **模型**: glm-4.7
- **定位**: 轻量任务
- **职责**:
  - 简单问答
  - 翻译
  - 简短总结
  - 日常对话
- **触发**: "你好"、"谢谢"、"翻译"、"简短"、"快速"

### 6. 🧠 深度思考者
- **模型**: qwen3-max-2026-01-23
- **定位**: 复杂推理
- **职责**:
  - 逻辑推理
  - 数学问题

  - 决策分析
- **触发**: "推理"、"逻辑"、"数学"、"战略"、"决策"、"复杂"

### 7. 📊 数据分析师
- **模型**: glm-5

- **职责**:

## 调度策略

### 一级分类 (按领域)
```
代码相关 → 代码专家
研究分析 → 研究员
创作写作 → 创意作家
简单问答 → 快速助手
复杂推理 → 深度思考者
数据处理 → 数据分析师
长文档/图片 → 麦克虾 (主)
```

### 二级选择 (按复杂度)
```
小代码任务 → qwen3-coder-next
大项目分析 → qwen3-coder-plus
简单查询 → glm-4.7
复杂分析 → kimi-k2.5
```

### 默认回退
- 无法分类 → 麦克虾 (qwen3.5-plus)
- 需要上下文 → 麦克虾 (100 万上下文优势)
